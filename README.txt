OpenClassRooms-devWeb - Projet 3
Code source du projet 3 de la formation développeur Web, intitulé "Dynamisez une page web avec des animations CSS".

Version 1.0.0

Le projet
La compagnie Ohmyfood!(compagnie fictive) est une entreprise de commande de repas haut de gamme en ligne. 
Après s'être développer à  New York elle veut maintenant se développer sur le marché parisien. 
Pour cela elle a fait appelle à mes compétences afin de créer la maquette de leur site dédier au réseau français.
Ces pages sont développées en HTML & CSS (Sass).

Cahier des charges
Cette intégration présentait un certain nombre d'exigeance à respecter :
-pas de frameworks
-le site doit être responsive (mobile, tablette et PC)
-Le header contient le logo (+ une flèche de retour à la landing page)
-un icon cliquable pour sauvegarder ses menus préférés.
-un loadinng spinner pour indiquer à l'utilisateur que la page se charge.
-des pages qui présentent le menu de chaque restaurant.
-La possibilité de sélectionner des éléments du menu (+ indicateur de sélection) 

Environnement

HTML
CSS (Sass)
